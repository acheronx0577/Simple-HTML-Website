# Step 2: Document Skeleton

Every HTML page requires a basic boilerplate skeleton of code.

## The Boilerplate Code

Copy this code exactly into your `index.html` file:

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Jane Doe - Profile</title>
  </head>
  <body>
    Jane Doe
  </body>
</html>
```

---

## Browser Output

Save the file and double-click `index.html` to open it in a browser. It should look like this:

![Skeleton Browser Render](images/step_02_render.png)

---

## Tag Breakdown

* <strong>`<!DOCTYPE html>`:</strong> Tells the browser that this is an HTML5 document.
* <strong>`<html>` and `</html>`:</strong> The root container for all code.
* <strong>`<head>` and `</head>`:</strong> Contains page metadata.
* <strong>`<meta charset="utf-8">`:</strong> Enforces UTF-8 encoding (ensuring clean character renders).
* <strong>`<title>` and `</title>`:</strong> Sets the browser tab text.
* <strong>`<body>` and `</body>`:</strong> Holds all visible screen content.
