# Step 3: Headings

HTML headings define the titles and hierarchy of your page, ranging from `<h1>` (largest) to `<h6>` (smallest).

## Headings Hierarchy

* <strong>`<h1>`:</strong> Used for the main title of the page (like a name).
* <strong>`<h2>`:</strong> Used for major sections (like "About Me").
* <strong>`<h3>` to `<h6>`:</strong> Used for smaller sub-sections.

---

## Code Example

Add these headings inside the `<body>` of your `index.html` file (replacing the plain "Jane Doe" text):

```html
<h1>Jane Doe</h1>
<h2>About Me</h2>
```

## Complete Step Code

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Jane Doe - Profile</title>
  </head>
  <body>
    <h1>Jane Doe</h1>
    <h2>About Me</h2>
  </body>
</html>
```
